# BookXchange
CS307 BookXchange


Running frontend:
- cd bookxchange_flutter
- flutter run


Running backend (seperate terminal):
- cd bookxchange_backend
- source venv/bin/activate
- python3 app.py

To install packages: 
- pip3 install -r requirements.txt
